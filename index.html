Image Toolkit Pro

Página oficial de soporte de la extensión.

Descripción

Image Toolkit Pro permite guardar imágenes desde páginas web en formato PNG, JPG o JPEG directamente desde el menú contextual del navegador.

Cómo usar

  1. Haz clic derecho sobre una imagen en una página web.
  2. Selecciona Guardar imagen como.
  3. Elige el formato: PNG, JPG o JPEG.
  4. Selecciona la ubicación de guardado.

Funciones principales

  • Guardar imágenes como PNG
  • Guardar imágenes como JPG
  • Guardar imágenes como JPEG
  • Conservar nombre original cuando sea posible
  • Configurar calidad JPG/JPEG
  • Redimensionar imágenes antes de guardar

Privacidad

Image Toolkit Pro procesa las imágenes localmente en el navegador. No recopila información personal ni envía imágenes a servidores externos.

Permisos utilizados

  • contextMenus: para agregar opciones al clic derecho
  • downloads: para guardar archivos
  • storage: para guardar preferencias del usuario
  • offscreen: para procesar imágenes en segundo plano
