Cómo usar

Una guía rápida para usuarios y revisores.

Pasos básicos

  1. Haz clic derecho sobre una imagen.
  2. Selecciona el formato que deseas guardar.
  3. Elige la ubicación y confirma la descarga.

Sobre la extensión

Image Toolkit Pro permite guardar imágenes desde cualquier página web en formato PNG, JPG o JPEG. Incluye opciones para definir calidad JPG/JPEG, conservar nombres de archivo y redimensionar imágenes antes de descargarlas. La conversión se realiza localmente en el navegador..

Sobre la extensión

Image Save Toolkit está pensada para ofrecer una forma directa y cómoda de guardar imágenes en formatos comunes, con una presentación simple y profesional.

Privacidad y permisos

Texto simple para explicar qué hace la extensión y qué no hace.

Privacidad

Image Save Toolkit procesa las imágenes localmente en el navegador. No recopila información personal ni envía imágenes a servidores externos para la conversión básica.

Permisos

  • contextMenus para las opciones del clic derecho
  • downloads para guardar archivos
  • storage para guardar preferencias
  • offscreen para el procesamiento interno
