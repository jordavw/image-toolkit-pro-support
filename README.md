# 🖼️ Image Save Toolkit

Extensión de navegador que permite guardar imágenes como **PNG, JPG o JPEG** directamente desde el menú contextual.

---

## 🚀 Características

* Guardar imágenes como:

  * PNG
  * JPG
  * JPEG
* Conversión local (sin subir archivos a servidores)
* Conserva el nombre original cuando es posible
* Configuración de calidad para JPG/JPEG
* Redimensionado opcional de imágenes
* Compatibilidad mejorada con sitios protegidos (fallback inteligente)

---

## 🧩 Cómo usar

1. Instala la extensión en tu navegador
2. Haz clic derecho sobre una imagen
3. Selecciona:

   * `Guardar imagen como PNG`
   * `Guardar imagen como JPG`
   * `Guardar imagen como JPEG`
4. Elige la ubicación de guardado

---

## ⚙️ Configuración

La extensión incluye opciones como:

* Calidad JPG/JPEG
* Modo de nombre de archivo
* Redimensionado de imágenes
* Descarga directa vs conversión

Puedes acceder desde:

> Configuración → Extensiones → Image Save Toolkit → Opciones

---

## 🔒 Privacidad

* ❌ No recopila datos personales
* ❌ No envía imágenes a servidores externos
* ✅ Todo el procesamiento se realiza localmente en el navegador

---

## 🛠️ Tecnologías utilizadas

* JavaScript (Vanilla)
* Chrome Extension API (Manifest V3)
* Offscreen API
* Canvas API
* Chrome Downloads API

---

## 📦 Instalación manual (modo desarrollador)

1. Clona este repositorio:

```bash
git clone https://github.com/tu-usuario/image-save-toolkit.git
```

2. Abre Chrome y ve a:

```
chrome://extensions/
```

3. Activa **Modo desarrollador**

4. Haz clic en **Cargar descomprimida**

5. Selecciona la carpeta del proyecto
